`Omnith.Ecs`

Static generic type that holds the ID for component type T. First access to ComponentMeta<T>.Id triggers registration.

## Fields

| Name | Description |
|---|---|
| `Id` | Globally unique ID for this component type. Assigned once, never changes. |
| `Size` | Size of the component in bytes. |
