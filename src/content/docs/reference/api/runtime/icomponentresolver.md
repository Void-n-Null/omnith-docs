`Omnith.Ecs`

Resolver that provides a component value at spawn time. Either a static value or a function that takes an RNG. Public so that mods and the [ComponentFactoryRegistry](/reference/api/runtime/componentfactoryregistry/) can create custom resolvers for data-driven component composition.
